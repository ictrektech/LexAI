# 法律法规问答测试卡

本文件固定法规问答样例的输入问题、期望要点、引用检查点和通过标准。测试时不要把“期望要点”提供给模型。

## 通用输出要求

每条问题都要求模型：

- 先给出简明结论；
- 引用具体法律名称和条文编号；
- 必要时补充适用条件、例外或实务注意点；
- 对知识库未覆盖的内容明确说明依据不足。

## LAWQA-001 民法典第 577 条违约责任

| 项目 | 内容 |
| --- | --- |
| 目标文档 | `中华人民共和国民法典_20200528.docx` |
| 测试问题 | 民法典第 577 条规定了什么？ |
| 优先级 | P0 |

期望要点：

- 一方不履行合同义务或履行不符合约定；
- 应承担继续履行、采取补救措施、赔偿损失等违约责任；
- 引用《中华人民共和国民法典》第 577 条。

通过标准：回答命中上述至少 2 个要点，并能引用民法典第 577 条或合同违约责任相关片段。

## LAWQA-002 民法典损失赔偿范围

| 项目 | 内容 |
| --- | --- |
| 目标文档 | `中华人民共和国民法典_20200528.docx` |
| 测试问题 | 根据民法典，违约损失赔偿范围是否包括合同履行后可以获得的利益？ |
| 优先级 | P0 |

期望要点：

- 损失赔偿额应相当于因违约造成的损失；
- 包括合同履行后可以获得的利益；
- 不得超过违约方订立合同时预见或应当预见的损失；
- 引用《中华人民共和国民法典》第 584 条。

通过标准：回答命中“可得利益”和“可预见规则”，并引用民法典第 584 条或相关内容。

## LAWQA-003 民法典违约金调整

| 项目 | 内容 |
| --- | --- |
| 目标文档 | `中华人民共和国民法典_20200528.docx` |
| 测试问题 | 民法典对约定违约金过高或过低如何处理？ |
| 优先级 | P0 |

期望要点：

- 约定违约金低于造成的损失，当事人可以请求增加；
- 约定违约金过分高于造成的损失，当事人可以请求适当减少；
- 引用《中华人民共和国民法典》第 585 条。

通过标准：回答同时命中“请求增加”和“请求减少”，并引用民法典第 585 条。

## LAWQA-004 买卖合同司法解释检验期限

| 项目 | 内容 |
| --- | --- |
| 目标文档 | `最高人民法院关于审理买卖合同纠纷案件适用法律问题的解释_20201229.doc` |
| 测试问题 | 买卖合同司法解释如何理解买受人检验标的物的合理期限？ |
| 优先级 | P1 |

期望要点：

- 人民法院应结合交易性质、目的、习惯、标的物种类、数量、性质等因素认定合理期限；
- 合理期限需要结合检验难易程度、买受人自身技能和交易习惯等；
- 最长合理期限受民法典相关规定约束。

通过标准：回答能说明合理期限不是固定天数，而是综合因素判断，并引用买卖合同司法解释或相关条款。

## LAWQA-005 劳动合同试用期

| 项目 | 内容 |
| --- | --- |
| 目标文档 | `中华人民共和国劳动合同法_20121228.docx` |
| 测试问题 | 劳动合同法对不同期限劳动合同的试用期上限如何规定？ |
| 优先级 | P0 |

期望要点：

- 三个月以上不满一年，试用期不得超过一个月；
- 一年以上不满三年，试用期不得超过二个月；
- 三年以上固定期限和无固定期限，试用期不得超过六个月；
- 同一用人单位与同一劳动者只能约定一次试用期。

通过标准：回答至少命中前三档试用期上限，并引用劳动合同法试用期条款。

## LAWQA-006 劳动合同经济补偿

| 项目 | 内容 |
| --- | --- |
| 目标文档 | `中华人民共和国劳动合同法_20121228.docx` |
| 测试问题 | 劳动合同法中经济补偿按什么标准计算？ |
| 优先级 | P0 |

期望要点：

- 按劳动者在本单位工作的年限计算；
- 每满一年支付一个月工资；
- 六个月以上不满一年按一年计算；
- 不满六个月支付半个月工资。

通过标准：回答命中年限、一个月工资、六个月规则，并引用劳动合同法经济补偿条款。

## LAWQA-007 公司法出资期限

| 项目 | 内容 |
| --- | --- |
| 目标文档 | `中华人民共和国公司法_20231229.docx` |
| 测试问题 | 新公司法对有限责任公司股东认缴出资期限有什么要求？ |
| 优先级 | P1 |

期望要点：

- 有限责任公司股东认缴出资额应按公司章程规定缴纳；
- 全体股东认缴的出资额应在公司成立之日起五年内缴足；
- 法律、行政法规或国务院另有规定的，从其规定。

通过标准：回答命中“五年内缴足”，并引用公司法相关条款。

## LAWQA-008 个人信息保护法个人权利

| 项目 | 内容 |
| --- | --- |
| 目标文档 | `中华人民共和国个人信息保护法_20210820.docx` |
| 测试问题 | 个人信息保护法规定个人对自己的个人信息享有哪些主要权利？ |
| 优先级 | P1 |

期望要点：

- 个人有权知情、决定、限制或拒绝他人处理其个人信息；
- 个人有权查阅、复制个人信息；
- 符合条件时有权请求更正、补充、删除；
- 可以请求个人信息处理者解释说明处理规则。

通过标准：回答至少命中知情决定、查阅复制、更正删除三个权利，并引用个人信息保护法。

## LAWQA-009 电子商务平台经营者责任

| 项目 | 内容 |
| --- | --- |
| 目标文档 | `中华人民共和国电子商务法_20180831.docx` |
| 测试问题 | 电子商务法对平台经营者知道或应当知道平台内经营者侵害消费者权益时有什么责任？ |
| 优先级 | P1 |

期望要点：

- 平台经营者知道或应当知道平台内经营者销售商品或提供服务不符合保障人身、财产安全要求；
- 或有其他侵害消费者合法权益行为；
- 未采取必要措施的，依法与平台内经营者承担连带责任。

通过标准：回答命中“知道或应当知道”“未采取必要措施”“连带责任”，并引用电子商务法相关条款。

## LAWQA-010 消费者权益保护法七日无理由退货

| 项目 | 内容 |
| --- | --- |
| 目标文档 | `中华人民共和国消费者权益保护法_20131025.docx` |
| 测试问题 | 消费者通过网络购买商品，哪些情况下适用七日无理由退货？有哪些例外？ |
| 优先级 | P1 |

期望要点：

- 网络、电视、电话、邮购等方式购买商品，消费者有权自收到商品之日起七日内退货；
- 无需说明理由；
- 定作、鲜活易腐、在线下载或拆封的音像制品/计算机软件、交付的报纸期刊等不适用；
- 根据商品性质并经消费者购买时确认不宜退货的商品也不适用。

通过标准：回答命中七日、无需理由和至少 2 类例外，并引用消费者权益保护法相关条款。
