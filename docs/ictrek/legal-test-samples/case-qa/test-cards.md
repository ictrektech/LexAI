# 裁判案例问答测试卡

本文件固定裁判案例问答样例的输入问题、期望要点、引用检查点和通过标准。测试时不要把“期望要点”提供给模型。

## 通用输出要求

每条问题都要求模型：

- 先给出简明结论；
- 说明目标案例中的案件事实；
- 涉及法律适用时，区分案件事实、裁判依据和法院观点/裁判理由；
- 给出可复核的来源引用；
- 对知识库未覆盖的内容明确说明依据不足。

## CASEQA-001 吴小秦案基本事实

| 项目 | 内容 |
| --- | --- |
| 目标文档 | `吴小秦诉陕西广电网络传媒（集团）股份有限公司捆绑交易纠纷案.md` |
| 测试问题 | 吴小秦案的基本事实是什么？请说明收费行为、金额构成和原告主张。 |
| 优先级 | P0 |

期望要点：

- 2012 年 5 月 10 日吴小秦缴纳 3 个月费用 90 元；
- 费用由数字电视基本收视维护费 75 元和数字电视节目费 15 元构成；
- 吴小秦主张广电公司构成搭售，侵害自主选择权；
- 广电公司与陕西省有线电视传输业务、节目集中播控的唯一经营地位有关。

通过标准：回答命中至少 3 个要点，并能引用吴小秦案或数字电视节目费相关片段。

| 项目 | 内容 |

| 目标文档 | `吴小秦诉陕西广电网络传媒（集团）股份有限公司捆绑交易纠纷案.md` |

| 优先级 | P0 |

## CASEQA-003 孙银山案十倍赔偿

| 项目 | 内容 |
| --- | --- |
| 目标文档 | `孙银山诉南京欧尚超市有限公司江宁店买卖合同纠纷案.md` |
| 测试问题 | 孙银山案中，消费者明知食品过期是否影响十倍赔偿？请说明事实、裁判结果和理由。 |
| 优先级 | P0 |

期望要点：

- 孙银山购买 15 包“玉兔牌”香肠，其中 14 包已过保质期；
- 过期香肠价值 558.6 元，十倍赔偿为 5586 元；
- 孙银山属于生活消费需要的消费者；
- 消费者是否明知食品不符合安全标准，不影响依法主张十倍赔偿。

通过标准：回答命中事实、金额、消费者身份和明知不影响赔偿中的至少 3 项，并引用孙银山案或食品安全法依据。
